.. py:function:: sma(close: polars.Expr, period: int = 20) -> polars.Expr

   Simple moving average over ``period`` observations.

   :param close: Price (or any series) to average.
   :param period: Window length.

   :returns: Rolling mean expression.


.. py:function:: ema(close: polars.Expr, period: int = 20) -> polars.Expr

   Exponential moving average with ``span = period``.

   :param close: Series to smooth.
   :param period: Span. The smoothing factor is ``2 / (period + 1)``.

   :returns: EWMA expression.


.. py:function:: wma(close: polars.Expr, period: int = 20) -> polars.Expr

   Linearly-weighted moving average.

   :param close: Series to smooth.
   :param period: Window length.

   :returns: Expression yielding the WMA. Recent observations have higher
             weight: weight ``i`` = ``i + 1`` for ``i in 0..period-1``.


.. py:function:: dema(close: polars.Expr, period: int = 20) -> polars.Expr

   Double exponential moving average: ``2 * EMA - EMA(EMA)``.

   :param close: Series to smooth.
   :param period: Span.

   :returns: DEMA expression.


.. py:function:: tema(close: polars.Expr, period: int = 20) -> polars.Expr

   Triple exponential moving average ``3*EMA - 3*EMA(EMA) + EMA(EMA(EMA))``.

   :param close: Series to smooth.
   :param period: Span.

   :returns: TEMA expression.


.. py:function:: midpoint(close: polars.Expr, period: int = 14) -> polars.Expr

   ``(rolling_max(close) + rolling_min(close)) / 2``.

   :param close: Price series.
   :param period: Window length.

   :returns: Midpoint expression.


.. py:function:: midprice(high: polars.Expr, low: polars.Expr, period: int = 14) -> polars.Expr

   ``(rolling_max(high) + rolling_min(low)) / 2``.

   :param high: Bar high.
   :param low: Bar low.
   :param period: Window length.

   :returns: Midprice expression.


.. py:function:: bbands_middle(close: polars.Expr, period: int = 20) -> polars.Expr

   Bollinger middle band — SMA of close.

   :param close: Price series.
   :param period: Window length.

   :returns: Rolling mean expression.


.. py:function:: bbands_upper(close: polars.Expr, period: int = 20, nbdev_up: float = 2.0) -> polars.Expr

   Bollinger upper band ``SMA + nbdev_up * std``.

   :param close: Price series.
   :param period: Window length.
   :param nbdev_up: Number of standard deviations above the SMA.

   :returns: Upper-band expression.


.. py:function:: bbands_lower(close: polars.Expr, period: int = 20, nbdev_dn: float = 2.0) -> polars.Expr

   Bollinger lower band ``SMA - nbdev_dn * std``.

   :param close: Price series.
   :param period: Window length.
   :param nbdev_dn: Number of standard deviations below the SMA.

   :returns: Lower-band expression.

.. py:function:: donchian_lower(low: polars.Expr, period: int = 20) -> polars.Expr
