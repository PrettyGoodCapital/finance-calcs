finance_calcs.alpha
===================

.. py:module:: finance_calcs.alpha

.. autoapi-nested-parse::

   Alpha / signal evaluation as polars expressions.

   These functions are designed to be composed inside ``group_by("date").agg(...)``
   to produce a cross-sectional information-coefficient time series, then
   aggregated across time with :func:`ic_ir` and friends.

.. py:function:: forward_returns(price: polars.Expr, periods: int = 1) -> polars.Expr

   Forward simple return over ``periods`` bars.

   :param price: Price series.
   :param periods: Look-ahead horizon in bars.

   :returns: Expression yielding ``price.shift(-periods) / price - 1``.

.. py:data:: information_coefficient

.. py:function:: ic_ir(ic: polars.Expr, *, window: int | None = None, period: finance_calcs._periods.PeriodLike | None = None, date: polars.Expr | None = None) -> polars.Expr

   IC information ratio — ``mean(ic) / std(ic)``.

   ``window=None`` → scalar; ``window=N`` → rolling IR over each
   trailing ``N``-observation window; ``period=...`` → per-bucket IR.


.. py:function:: hit_rate(signal: polars.Expr, fwd: polars.Expr) -> polars.Expr

   Fraction of observations where ``sign(signal) == sign(fwd)``.

   :param signal: Signal series.
   :param fwd: Forward return series.

   :returns: Scalar mean expression in ``[0, 1]``.


.. py:function:: ic_summary_stats(ic: polars.Series) -> dict[str, float]

   Summary statistics of an IC time series.

   :param ic: IC time series as a polars Series.

   :returns: Dict with ``mean``, ``std``, ``ir``, ``t_stat``, ``pct_positive``,
             ``n``. ``t_stat`` is ``ir * sqrt(n)``.
