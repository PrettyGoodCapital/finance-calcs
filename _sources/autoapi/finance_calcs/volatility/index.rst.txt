.. py:function:: parkinson_vol(high: polars.Expr, low: polars.Expr, period: int = 20) -> polars.Expr

   Parkinson high-low range volatility estimator.

   .. math::
       \\hat{\\sigma}^2 = \\frac{1}{4 \\ln 2} \\cdot \\overline{\\left(\\ln(H/L)\\right)^2}

   :param high: Bar high.
   :param low: Bar low.
   :param period: Window length.

   :returns: Per-period volatility expression (rolling).


.. py:function:: garman_klass_vol(open_: polars.Expr, high: polars.Expr, low: polars.Expr, close: polars.Expr, period: int = 20) -> polars.Expr

   Garman-Klass OHLC volatility estimator.

   .. math::
       \\hat{\\sigma}^2 = \\overline{\\tfrac{1}{2}(\\ln H/L)^2 - (2\\ln 2 - 1)(\\ln C/O)^2}

   :param open_: Bar open.
   :param high: Bar high.
   :param low: Bar low.
   :param close: Bar close.
   :param period: Window length.

   :returns: Per-period GK volatility expression (rolling).


.. py:function:: rogers_satchell_vol(open_: polars.Expr, high: polars.Expr, low: polars.Expr, close: polars.Expr, period: int = 20) -> polars.Expr

   Rogers-Satchell drift-independent volatility.

   .. math::
       \\hat{\\sigma}^2 = \\overline{\\ln(H/C)\\ln(H/O) + \\ln(L/C)\\ln(L/O)}

   :param open_: Bar open.
   :param high: Bar high.
   :param low: Bar low.
   :param close: Bar close.
   :param period: Window length.

   :returns: RS volatility expression (rolling).


.. py:function:: yang_zhang_vol(open_: polars.Expr, high: polars.Expr, low: polars.Expr, close: polars.Expr, period: int = 20, k: float | None = None) -> polars.Expr

   Yang-Zhang volatility — minimum-variance combination of overnight,
   open-to-close, and Rogers-Satchell drift-independent components.

   :param open_: Bar open.
   :param high: Bar high.
   :param low: Bar low.
   :param close: Bar close.
   :param period: Window length.
   :param k: Weight on open-to-close variance. Defaults to
             ``0.34 / (1.34 + (period+1)/(period-1))``.

   :returns: YZ volatility expression (rolling).
