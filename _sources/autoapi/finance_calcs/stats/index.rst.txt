finance_calcs.stats
===================

.. py:module:: finance_calcs.stats

.. autoapi-nested-parse::

   Statistical-validity calculations.

   Combines polars-expression metrics (skew/kurtosis/stability) with
   ``pl.Series``-level helpers for bootstrap and probabilistic-Sharpe
   analyses that need numerical work outside the expression engine.

.. py:function:: stability_of_timeseries(returns: polars.Expr) -> polars.Expr

   Coefficient of determination of cumulative log returns vs time.

   Implements pyfolio's ``stability_of_timeseries`` — fit
   :math:`y_t = a + b \cdot t` to the log-equity curve and return
   ``R^2``. Closer to 1 means more linear (steady) growth.

   :param returns: Periodic returns (not log).

   :returns: Scalar ``R^2`` expression.


.. py:function:: common_sense_ratio(returns: polars.Expr) -> polars.Expr

   :param returns: Periodic returns expression.

   :returns: Scalar expression.
