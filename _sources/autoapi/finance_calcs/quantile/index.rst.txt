finance_calcs.quantile
======================

.. py:module:: finance_calcs.quantile

.. autoapi-nested-parse::

   Quantile and signal-quantile analysis as polars expressions.

   Most functions are intended to be evaluated cross-sectionally inside
   ``group_by("date").agg(...)`` or ``with_columns(... .over("date"))``.

.. py:function:: assign_quantile(signal: polars.Expr, n_quantiles: int = 5) -> polars.Expr

   Assign integer quantile labels ``0..n_quantiles-1`` to ``signal``.

   :param signal: Signal series. Nulls produce null labels.
   :param n_quantiles: Number of quantile buckets.

   :returns: Integer expression in ``[0, n_quantiles - 1]``. Higher signal
             values map to higher labels.


.. py:function:: rank_normalize(signal: polars.Expr) -> polars.Expr

   Cross-sectional rank scaled to ``[-0.5, 0.5]``.

   :param signal: Signal series.

   :returns: Expression with mean zero and bounded support.


.. py:function:: zscore(signal: polars.Expr) -> polars.Expr

   Cross-sectional z-score: ``(x - mean) / std``.

   :param signal: Signal series.

   :returns: Z-score expression.


.. py:function:: winsorize(signal: polars.Expr, cutoff: float = 3.0) -> polars.Expr

   Clip values to ``mean ± cutoff * std``.

   :param signal: Signal series.
   :param cutoff: Number of standard deviations. Must be positive.

   :returns: Clipped expression.


.. py:function:: long_short_spread(returns: polars.Expr, quantile: polars.Expr, upper: int, lower: int) -> polars.Expr

   Top-quantile mean return minus bottom-quantile mean return.

   Use inside ``group_by("date").agg(...)``::

       df.group_by("date").agg(
           long_short_spread(pl.col("ret"), pl.col("q"), upper=4, lower=0)
           .alias("ls"),
       )

   :param returns: Forward return series.
   :param quantile: Integer quantile label series.
   :param upper: Long quantile label.
   :param lower: Short quantile label.

   :returns: Scalar expression.


.. py:function:: quantile_changed(quantile: polars.Expr) -> polars.Expr

   Boolean expression: ``quantile != quantile.shift(1)``.

   Use inside ``... .over("asset")`` to compute per-asset turnover
   flags. Aggregate by date to get the fraction of names that changed
   quantile.

   :param quantile: Integer quantile label series.

   :returns: Boolean expression. The first observation is null/false.
